# 计网黑话🥲

IP(**Internet Protocol**) :互联网协议

TCP(**Transmit Control Protocol**): 传输控制协议

UDP(**User Datagram Protocol**): 用户数据报协议

HTTP(**Hypertest Transfer Protocol**): 超文本传输协议

URL(**Uniform Resource Locator**): 统一资源定位符,即用户在浏览器中输入的网址

DNS(**Domain Name System**): 域名系统，把**域名**（如 www.example.com）翻译成**IP 地址**（如 192.0.2.1）的系统

**DNS 的结构**

​	•	**顶级域名 (TLD)**：如 .com、.org、.net。

​	•	**二级域名**：如 example。

​	•	**子域名**：如 www。

​	•	**DNS 服务器**：包括递归服务器、根服务器和权威服务器等，帮助解析域名。
